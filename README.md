# AI陪伴养成

像真人一样的AI陪伴，会聊天、会生气、会拉黑，越聊越懂你。

## 功能特性

-   **拟人化AI**：接入真实AI，像真人一样聊天
-   **新联系人场景**：刚匹配上，互相不认识，有警惕心
-   **情绪系统**：会开心、生气、吃醋、撒娇
-   **拉黑/删除**：被骚扰会骂人、拉黑、删除好友
-   **学习进化**：根据对话不断学习，越来越像真人
-   **密钥保管**：API密钥自动保存到本地，下次启动无需输入

## 快速开始

### 安装

```bash
git clone https://github.com/TT-XL/AI-personality-project.git
cd AI-personality-project
npm install
```

### 运行

```bash
# 双击 start.bat 启动
# 或
npm run dev:ai
```

## 核心功能

### 1. 新联系人场景

- 刚匹配上，互相不认识
- 保持警惕心，不会轻易相信
- 会质疑对方意图
- 随着聊天增多，慢慢熟悉

### 2. 情绪系统

| 情绪 | 表现 |
|------|------|
| 开心 | 话多、发表情包、会撒娇 |
| 生气 | 话少、会骂人、冷淡 |
| 吃醋 | 阴阳怪气、试探 |
| 撒娇 | 说"哼"、"不理你了" |
| 无聊 | 找话题、吐槽 |
| 心情不好 | 不想说话、敷衍 |

### 3. 拉黑/删除系统

| 反感程度 | 行为 |
|----------|------|
| 0-10 | 正常聊天 |
| 10-30 | 已读不回 |
| 30-50 | 可能骂人 |
| 50-80 | 骂人或拉黑 |
| 80-100 | 拉黑或删除 |

### 4. 学习进化系统

- 记录成功回复
- 学习用户模式
- 学习情绪状态
- 根据对话调整行为

### 5. 密钥保管功能

- API密钥自动保存到本地
- 下次启动自动加载
- 不用每次输入密钥
- 安全存储，不会上传
